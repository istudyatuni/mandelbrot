# Mandelbrot set

## Try online

https://istudyatuni.github.io/mandelbrot

## Build WASM

You need rust and cargo (for installing build tool). Install `wasm-pack`:

```bash
cargo install wasm-pack
```

Build:

```bash
./scripts/build-wasm.sh
# or
yarn build:wasm
```

## Running locally

```bash
yarn dev
```
